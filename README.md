# Moho inversion program 
`mohoinv_development_7.py` is a program soving moho relief inversion problem using Uieda & Barbosa 2017 strategy.
We rewrite the program with new fatiando a terra [[text](https://www.fatiando.org/)](https://www.fatiando.org/) which provided super efficient tessroid gravity forward modeling calculations.

`test.ipynb` gives a little usage guidence.

See more detail 👉 [https://neustr-ynu.github.io/software/pages/moho_inversion_tutorial.html](https://neustr-ynu.github.io/software/pages/moho_inversion_tutorial.html)
